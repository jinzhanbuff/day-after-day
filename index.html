2030年未来就业技能

世界经济论坛《未来就业报告》

新兴技能 (Emerging Skills)

现阶段不算核心，但未来需求将大幅增长

  • ✓ 网络与网络安全
  • ✓ 环境管理
  • ✓ 设计与用户体验

核心技能 (Core Skills)

目前核心且未来重要性将继续提升

  • ✓人工智能与大数据
  • ✓技术素养
  • ✓好奇心与终身学习
  • ✓创造性思维
  • ✓韧性、灵活性与敏捷性
  • ✓领导力与社会影响力
  • ✓分析性思维
  • ✓动机与自我认知
  • ✓同理心与积极倾听
  • ✓客户服务与服务导向
  • ✓资源管理与运营管理
  • ✓可靠性与细致关注

不受关注技能 (Out-of-focus)

当前不重要且未来需求下降

  • ✓ 编程 (相对较低)
  • ✓ 多语言能力
  • ✓ 营销媒体
  • ✓ 阅读写作数学
  • ✓ 体力技能等

稳定技能 (Steady Skills)

目前为核心，但未来使用不会显著增加

  • ✓ 系统思维
  • ✓ 教学与指导
  • ✓ 质量控制
